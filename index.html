Rahu Yama Time

Ruby gem used to fetch Raahukaalam, Gulikai and/or Yamakandam for either a day or a full week.

Raahukaalam, Gulikai and Yamakandam are well known timing schedules in Hindu religion. They denote the good and bad time and vary for every day of a week and are constant for all weeks of year. To fetch this timing either per day or as a week, one can make use of this gem in their application to fetch the required details.

The gem code is available online here and the usage steps can be known from the gem's home page.

Details about the gem:
  • Programming language: Ruby
  • Year: 2013
  • Lines of code: 70
  • Time taken to develop: Approx. 2 days at leisure

Chess

A 2-player Chess game designed completely using C++.

Chess is a 2 player game designed by me using C++ programming language. The controls are only through keyboard. The coin is chosen by entering the position of the coin and the final position for the coin is chosen as the same. The program validates if the move is a valid or invalid move by using a miniature version of chess board.

The miniature version of the chess board is a 8x8 array where black coins are represented using lower case alphabets a-h and white coins using upper case. This minitaure version along with other important variables' values can be stored into a file, there by saving a game's progress and continuing to play it later.

Details about the application:
  • Programming language: C++
  • Year started: 2007
  • Lines of code: 1641
  • Time taken to develop: Stopped developing after 2 years. Almost completed.

Prince School and College Bell Project

This project was created by self-interest to computerize the bell ringing process at my school and college.

This project was developed twice. Once during my school days and later during college.

During school days, I did not have the knowledge to get system time using C/C++ program. So I took MS Office's Power Point and used Rehearse Timings option to achieve it. The issue with this approach was that, a consistent power back up was required to keep the slide show running all throught the day. If power goes off even for a second, then the slide show will get cut and everything will go out of sync. Hence it was rejected.

During the end of my college second year (2009 mid), I developed the same application using Applet. Applet program fetched time from the system and played a bell sound when the exact time arrived. Also the prayer song and other songs such as National Anthem were programmed to be played at correct timings. Separate programs were written for bell ringing during semester examinations.

Details about the application:
  • Programming language: Java Applet
  • Year: 2009
  • Time taken to develop: Approx. 1 month in leisure time.

* Currently the applet programs are not running on browser as the support has been terminated. The actual intention of this project was to embed the applet in HTML pages and use them.
